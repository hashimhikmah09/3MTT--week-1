# 3MTT WEEK 2 TASK

A simple **personal page** built with **HTML5** and **external CSS**.
This project introduces the structure of a basic personal webpage that showcases a developer’s background, skills, mission, and contact information.

The page demonstrates **semantic HTML structure**, clean layout organization, and separation of content and styling using an external stylesheet.

---

## Features

* Navigation bar with links to page sections
* Hero section introducing the page
* About section with profile image
* Skills list
* Mission statement
* Contact section with email link
* Footer with copyright information
* Clean layout using external CSS

---

## Technologies Used

* HTML5
* CSS3
* Semantic HTML structure

---

## Project Structure

```
project-folder
│
├── index.html
├── style.css
└── hikmah.jpeg
```

**index.html** – Contains the webpage structure
**style.css** – Contains all styling for the page
**hikmah.jpeg** – Profile image used in the About section

---


## How to Run the Project

1. Download or clone the project folder.
2. Open the folder in your computer.
3. Double-click **index.html**.
4. The page will open in your web browser.
